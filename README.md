# Travel Companion App

An android studio mobile application used to convert units of measurements when travelling internationally. Created by Phillip Pavlides for SIT305 at Deakin University

## Installation
*   Open Git Bash
*   Clone the repository: `git clone https://github.com/PhillipPav/Travel_Companion_App`

## Usage
The program allows users to select units of measurement from a small list and convert them between eachother. Categories include Currency, Fuel/Distance & Efficiency and Temperature
*   Select a unit of measuremnt from the ***first*** dropdown menu to convert ***from***
*   Select a unit of measurment from the ***second*** dropdown menu to convert ***to***
*   Press calculate button to show the conversion output
